# BrickGame — Tetris на C (ncurses)

Системный Tetris на C с разделением на библиотеку логики и терминальный UI (ncurses), конечными автоматами и >80% покрытием unit-тестами (check). Под Linux.

## Возможности
- FSM для игровой логики: генерация фигур, столкновения, линии, уровни
- Сохранение рекордов между запусками, динамическая скорость
- Makefile: `all`, `install`, `test`, `gcov_report`

## Сборка и запуск
```bash
make
./bin/tetris
